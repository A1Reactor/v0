# A1 Reactor: Iteration-First Generation

## Abstract
A1 Reactor is a pipeline prototype designed to turn generation into iteration.
It frames video generation as a sequence of deterministic steps with explicit artifacts.

## Motivation
Creators need control: the ability to keep what works and fix what doesn't.
This repository provides the orchestration, interfaces, and tooling needed for that workflow.

## Design principles
- Deterministic file layout
- Provider abstraction
- Minimal API surface
- Metrics and diff tooling
- Small, composable modules

## Future work
- Multi-shot storyboards
- Better alignment and blending
- Interactive UI
- Richer metrics
